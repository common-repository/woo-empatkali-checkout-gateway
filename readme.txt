Add Empatkali payment gateway to your website checkout. No Coding Required.

== Description ==
= Overview =

Empatkali payment gateway plugin will allow you to purchase products on your website using this during checkout.

== Installation ==
> 1. Sign in to your WordPress site as an administrator.
> 2. Make sure that you install WooCommerce plugin first before installing this.
> 3. In the main menu go to Plugins -> Add New.
> 4. Search for Empatkali and click install.
> 5. In the main menu go to WooCommerce -> Settings then choose "Payments" tab.
> 6. Look for "Empatkali Payment Gateway" as the payment method then supply required input fields; **Note:** make sure that you set the plugin to "enabled".
> 7. That's it. You are now ready to start Empatkali as one of your payment method during checkout.

== Frequently Asked Questions ==
= How do I use this Empatkali plugin =
In your WooCommerce payment settings, input the required fields such as the username and password and make sure that it is working.
If all is well, then you should see Empatkali as one of your payment gateway during checkout.
